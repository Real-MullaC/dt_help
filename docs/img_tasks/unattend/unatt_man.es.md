# Unattended answer file manager

<p align="center">
    <img src="../../res/img_tasks/unattend/unatt_man.png" />
</p>

The unattended answer file manager lets you manage unattended answer files for your project. You can access this manager using 2 methods:

- Go to Tools > Unattended answer file manager
- Right-click Unattended answer files in the project tree view and go to Unattended answer files > Manage

*This tool is available in DISMTools 0.5.1 and later.*

## Usage

Simply select your unattended answer file and choose the action to perform by clicking the buttons at the bottom of the window.
