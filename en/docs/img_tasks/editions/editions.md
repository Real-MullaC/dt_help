# Image edition operations

By performing image edition operations, you can upgrade a Windows image to a higher edition and set the product key. You can also get the current edition and the target editions you can upgrade to.

These tasks are available by going to Commands > Windows editions.

## Available tasks

- [Upgrading an image to a higher edition](../set_edition)
- [Changing the product key](../set_productkey)
