# Setting keyboard layered drivers

<p align="center">
    <img src="../../../res/img_tasks/langs/set_layereddriver.png" />
</p>

With DISMTools you can configure keyboard layered drivers for users of Japanese and Korean keyboards.

*This action is supported on DISMTools 0.5.1 and newer.*

## Usage

Configure keyboard layered drivers using one of the following types and click OK:

- PC/AT Enhanced Keyboard (101-Key)
- Korean PC/AT 101-Key Compatible Keyboard - Microsoft Natural Keyboard (Type 1/2/3)
- Korean Keyboard (103/106 Key)
- Japanese Keyboard (106/109 Key)
