# Windows Image Explorer

<p align="center">
	<img src="../../../res/img_tasks/mgmt/WIM_Explorer.png" />
</p>

If you want to get information about the files and folders of any Windows image, you can use the **Windows Image Explorer** to achieve the job.

<i>This action is supported on DISMTools 0.5 and newer.</i>

You can find it as part of DISMTools, but you can also use it separately. Check out the [Windows Image Explorer repository](https://github.com/CodingWonders/WIM-Explorer), from which you can download the Windows Image Explorer.

## Usage

1. Go to Commands > Image management > List files and directories in image...
2. Browse

For more usage information, check out the aforementioned repository.
